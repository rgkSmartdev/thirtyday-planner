30-day LeetCode plan mapped to TUF DSA Sheet (from “easy-level” → solid medium)
This is a TUF-pattern-first plan: you learn the pattern from TUF, then solve the closest LeetCode set. Every day includes Must-do (core) and Optional (if time). Target: by Day 30 you should be consistently solving mediums under time pressure.

Daily DSA routine (2.5–3h)

15–20m: read/watch TUF for the pattern
60–90m: Must-do problems (timed; aim 25–35 min per medium)
30–45m: Optional OR re-solve 1 earlier problem without looking
15m: mistake log (pattern, bug type, what to remember)
I’m listing exact LeetCode titles so you can search quickly.
